ðŸ“‹ Panduan Penggunaan ViewTable

Untuk Table Biasa:

  • <table class="viewtable" data-viewtable> - Auto mode
  • Gunakan class="viewtable-detail" untuk kolom secondary
  • Default: 3 kolom pertama yang bukan detail

Untuk Table Kompleks (colspan/rowspan):

  • Selalu gunakan data-mobile-cols="col1,col2,col3"
  • Nama kolom harus exact match dengan <th> text
  • Urutan di data-mobile-cols menentukan urutan di mobile

Best Practices:

  • Gunakan text-sm, text-xs untuk text kecil
  • Gunakan font-semibold untuk emphasis
  • Gunakan badges untuk status/numerik penting
  • Max 4 kolom visible di mobile
  • Test di berbagai ukuran layar
